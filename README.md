# snake-3D

This is a snake 3D game

* You can move either with WASD or the arrow keys
* You need the following librarys to run the game:
    1. numpy
    2. scipy
    3. matplotlib
    4. pyopengl
    5. glfw
    6. ipython
    7. pillow
    8. jupyter

* The game runs with the command "python snake.py N" where N is the grid size
* Have fun :)
